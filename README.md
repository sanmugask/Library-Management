# Library Management System Using C and Linked Lists

Overview

The Library Management System is a console-based application developed in C using Linked Lists and File Handling. The system automates library operations such as managing books, issuing and returning books, and maintaining borrower records. It provides an efficient way to organize and track library resources while demonstrating the practical implementation of Data Structures.

Objectives

* Manage library book records efficiently.
* Perform CRUD operations on book records.
* Implement Linked List Data Structure.
* Handle book issue and return transactions.
* Store data permanently using files.
* Provide a user-friendly menu-driven interface.

Features

1. Add New Book
2. View All Books
3. Search Book by ID
4. Search Book by Title
5. Update Book Details
6. Remove Book Record
7. Issue Book
8. Return Book
9. View Issued Books
10. Save Records to File
11. Load Records from File
12. Exit

Software Requirements

* Ubuntu Linux
* GCC Compiler
* Make Utility
* VS Code / Vim Editor

Conclusion
